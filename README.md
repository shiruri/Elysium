## Documentation



Elysium is a basic based programming languange
Made using Java 

It consists of

LEXER - Splits words into Tokens

PARSER - Responsible for making meaning of the tokens

It can perform basic functions and supports the following


1. STRINGS
2. EXPRESSION 
9. CONCATINATIONS
10. INPUT
11. NUMBERS
12. VARIABLES
13. IF STATEMENT

1. Ouput:
```
print "String"
print NUM
print Expression

```
Example Usage : 
```
print "Hello Word"
print 67
print 1 + 1
```

2. Input:

```
input keyword <"String" message > <Variable>
```
Example Usage :
```
input "Enter name: " @name
```

3. Variables
```
@symbol = data
```
Example Usage:
```
@name = "Name"

@sum = 1 + 1
```
